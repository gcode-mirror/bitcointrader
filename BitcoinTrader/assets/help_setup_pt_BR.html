Configurações

Para ser capaz de utilizar esse aplicativo, você tem que conectá-lo à sua conta do Mt.Gox. Existem duas possibilidades disponíveis que você pode encontrar na \"Security Center\" do site MtGox.com (https://mtgox.com/security). Em ambos os casos você precisa habilitar as funções \"Get Info\" e \"Trade\".

Clique no QR code para escanear a sua chave de \"Acesso à API de negociações\" Application Access: Gera um qr-code que pode ser escaneado dentro desse applicativo.

Advanced API Key Creation: se você não quer gerar um qr-code no site do Mt.Gox, você pode criar um par de chaves e entrar com ambas as chaves nos campos de texto.

Se você não tem certeza, se esse não é o aplicativo certo pra você, você pode iniciar uma demonstração clicando no botão!
